T-Invest Portfolio Grid Bot v1.0
1. Назначение системы

Портфельный мультиактивный торговый робот для платформы T-Invest на Python.

Основные задачи:

Автоматическая торговля по сеточной стратегии
Работа с несколькими счетами
Работа с несколькими инструментами
Работа по индексам и пользовательским спискам
Управление капиталом
Сбор статистики
Бэктестирование и оптимизация стратегии
2. Ограничения версии 1.0
Только Spot торговля
Только Long позиции
Без шортов
Без маржинальной торговли
Без заемных средств

6. Торговая логика
Построение сетки

При создании новой сетки:

Берем историю
Находим MIN LOW
Находим MAX HIGH
Делим диапазон на уровни
Фиксируем сетку до завершения

Пользователь задает:

Количество уровней
Период истории
Вход

После достижения уровня:

Включаем трейлинг входа

Трейл:

0.5%

Подтверждение разворота:

0.15%

После подтверждения:

Лимитная покупка
на 0.15% выше рынка
Выход

Считаем:

Комиссия покупки
+
Комиссия продажи
+
Минимальная прибыль 0.3%
+
Запас 0.15%

После достижения цели:

Включаем трейлинг выхода

Трейл:

0.5%

Продажа:

Лимиткой ниже рынка на 0.15%

10. Управление капиталом

Перед запуском:

Считаем необходимый депозит

По всем активам.

Недостаток средств

Разрешается запуск с нехваткой капитала.

Пользователь принимает риск.

Если денег нет:

WAITING_FOR_FUNDS

После появления средств:

Повторная проверка условий
Новый трейлинг
Покупка
11. Режимы остановки

Только наблюдение
Без действий
EMERGENCY_CLOSE
Отмена заявок
Продажа всех позиций
Остановка работы

Продажа:

Лимитка на 0.5% ниже рынка
